About
Waachak (meaning reader in Hindi language) is a free RSS Reader service. Enjoy!

Application
This RSS reader is using AngularJS and .Net Web APIs with SQL Server as the backend database. The complete source code can be downloaded free at Github here.

Why another reader?
Just because... I was playing with AngularJS and ended up building this application. It has been a lot of fun!
